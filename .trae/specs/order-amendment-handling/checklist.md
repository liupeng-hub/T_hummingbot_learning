# Checklist

## 入场单事件处理

- [x] FILLED 状态处理（成交）
- [x] CANCELED 状态处理（取消）
- [x] AMENDMENT 状态处理（修改价格、数量）
- [x] EXPIRED 状态处理（过期）
- [x] TRADE_PREVENT 状态处理（STP 触发）

## 条件单事件处理

- [x] NEW 状态处理（新建条件单）
- [x] FILLED 状态处理（条件单成交）
- [x] CANCELED 状态处理（条件单取消）
- [x] EXPIRED 状态处理（条件单过期）
- [x] REJECTED 状态处理（条件单被拒绝）
- [ ] TRIGGERED 状态处理（条件单已触发）- 可选

## 入场单修改处理

- [x] 价格修改 → 更新止盈止损价格
- [x] 数量修改 → 更新本地数量
- [x] 同时修改价格和数量

## 条件单修改处理

- [x] 止盈单价格修改 → 更新本地止盈价
- [x] 止损单价格修改 → 更新本地止损价

## 日志和通知

- [x] 条件单修改日志
- [x] 入场单过期通知
- [x] 条件单过期通知
- [x] 条件单拒绝通知
