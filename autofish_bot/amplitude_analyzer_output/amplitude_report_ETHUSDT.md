## 算法说明

### 振幅计算
```
振幅 = (high - low) / open × 100
```

### 预期收益计算
```
预期收益 = 振幅 × 杠杆 × 概率
```

### 权重计算
```
权重 = 振幅 × 概率^(1/d)
```
- d=0.5: 幂次β=2，权重集中在前几层（激进）
- d=1.0: 幂次β=1，权重分布均匀（保守）
