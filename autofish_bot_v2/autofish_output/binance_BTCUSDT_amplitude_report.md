**策略说明**: 入场价格策略用于计算 A1 订单的入场价格。

- **fixed**: 固定网格间距，入场价格 = 当前价格 × (1 - 网格间距)
- **atr**: 基于 ATR 动态计算，适应市场波动性
- **bollinger**: 布林带下轨入场，适用于震荡市场
- **support**: 支撑位入场，适用于趋势市场
- **composite**: 综合多种技术指标

## 算法说明

### 振幅计算
```
振幅 = (high - low) / open × 100
```

### 预期收益计算
```
预期收益 = 振幅 × 杠杆 × 概率
```

### 权重计算
```
权重 = 振幅 × 概率^(1/d)
```
- d=0.5: 幂次β=2，权重集中在前几层（激进）
- d=1.0: 幂次β=1，权重分布均匀（保守）
