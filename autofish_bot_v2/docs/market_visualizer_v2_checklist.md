## 测试用例

### 测试用例 1：基本流程测试
```
1. 打开 Web 页面
2. 点击"新建测试"
3. 填写参数：
   - 名称: 测试1
   - 交易对: BTCUSDT
   - 时间范围: 2023-01-01 ~ 2023-03-31
   - 算法: dual_thrust
   - 参数: k2_down_factor=0.6, down_confirm_days=1
4. 点击"执行测试"
5. 等待执行完成
6. 检查 K线图显示
7. 检查统计数据
8. 检查区间表格
```
**预期结果**:
- [ ] 测试创建成功
