# test_manager.py CLI 重构计划

## 任务概述
重构 `test_manager.py` 的 CLI 命令，移除过时的"计划-场景-用例"结构，简化为测试用例的 CRUD 操作。

## 当前 CLI 命令（过时）
- `create-plan` - 创建测试计划
- `list-plans` - 列出测试计划
- `show-plan` - 查看测试计划
- `run-plan` - 执行测试计划
- `summary` - 生成测试汇总
- `compare` - 生成对比报告
- `history` - 查看测试历史
- `query-results` - 查询测试结果
- `export` - 导出测试报告
- `stats` - 查看统计数据
- `serve` - 启动Web API服务

## 新 CLI 命令设计

### 1. `create-case` - 创建测试用例
参考 `binance_backtest.py` 的参数设计：

```bash
python test_manager.py create-case \
    --symbol BTCUSDT \
    --date-range 20200101-20260310 \
    --name "测试用例名称" \
    --description "描述" \
    --amplitude-params '{"grid_spacing": 0.01, ...}' \
    --market-params '{"algorithm": "dual_thrust", "dual_thrust": {...}}' \
    --entry-params '{"strategy": "atr", "atr": {...}}' \
    --timeout-params '{}'
```

**参数说明：**
| 参数 | 必选 | 说明 |
|------|------|------|
| --symbol | 是 | 交易对 |
| --date-range | 是 | 时间范围 (yyyymmdd-yyyymmdd) |
| --name | 否 | 用例名称 |
| --description | 否 | 用例描述 |
| --amplitude-params | 否 | 振幅参数 (JSON) |
| --market-params | 否 | 行情算法参数 (JSON) |
| --entry-params | 否 | 入场价格策略参数 (JSON) |
| --timeout-params | 否 | 超时参数 (JSON) |

### 2. `list-cases` - 列出测试用例
```bash
python test_manager.py list-cases [--symbol BTCUSDT] [--status active]
```

### 3. `show-case` - 查看测试用例详情

```bash
python test_manager.py show-case --case-id <case_id>
```

### 9. `export` - 导出测试报告（保留）

```bash

```
